# 智能学习预警服务系统 — 功能结构介绍

## 一、系统概述

**智能学习预警服务系统** 是一套面向高校的智能化学生学业管理系统，基于 **Spring Boot + Spring Cloud Alibaba** 微服务架构开发。系统覆盖 **学生、教师（班主任）、辅导员、管理员、学院管理员** 五类角色，围绕 **成绩管理 → 智能预警 → 帮扶计划 → 数据分析** 的核心业务闭环，并集成 **通义千问 AI 大模型** 提供智能分析能力。

## 六、核心引擎

### 6.1 学业预警引擎

- **预警等级：** 低、中、高、严重 四级
- **触发条件：** 单科不及格、多科不及格、GPA 偏低、GPA 严重偏低、成绩下降趋势、自定义规则
- **扫描策略：** 全量扫描（每天 02:00）+ 增量扫描（每 30 分钟），自动去重
- **联动动作：** 预警生成后自动通知学生和相关辅导员，可触发帮扶计划分配

### 6.2 AI 智能分析引擎

- **模型：** 通义千问 qwen-plus
- **能力：** 成绩分析、风险预测、趋势分析、学习建议、预警建议、帮扶计划智能生成、智能对话
- **双引擎架构：** 大模型 + 规则引擎兜底，API 不可用时自动降级

### 6.3 消息通知系统

- **通知类型：** 预警生成自动通知、帮扶分配通知、申诉处理通知、广播消息、定向私信
- **推送方式：** WebSocket 实时推送 + 消息列表持久化
- **消息状态：** 未读/已读、批量操作、清空

## 八、技术栈

| 层级 | 技术选型 |
|------|----------|
| 后端框架 | Spring Boot 3.2.5 + Spring Cloud 2023.0.3 |
